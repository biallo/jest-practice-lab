# Jest Practice Lab

一个 Jest 的学习项目。

## 在线预览

[https://biallo.github.io/jest-practice-lab/](https://biallo.github.io/jest-practice-lab/)

## 运行

```shell
npm install
npm run dev
```

## 测试

运行全部 Jest 测试：

```shell
npm test
```

开发时可以使用 watch 模式：

```shell
npx jest --watch
```

只运行某个测试文件：

```shell
npx jest src/hooks/useCompletedLessons.test.tsx
```


## 编译和构建

```shell
npm run build
```
